    Groebner.jl

    Groebner.jl is a package for fast and generic Gröbner bases computations based on Faugère's F4 algorithm [1] written in pure Julia.

    Installation

    To install Groebner.jl, use the Julia package manager:

    using Pkg
Pkg.add("Groebner")

    See Interface for a description of all exported functions. Find a quick introduction to Groebner bases and a couple of interesting use cases in Tutorials. Meanwhile, below are simple examples.

    Examples

    Currently, polynomials from AbstractAlgebra, DynamicPolynomials, and Nemo are supported as input.

    with AbstractAlgebra

    Let's import AbstractAlgebra, create a system over a finite field

    using AbstractAlgebra

R, (x, y) = PolynomialRing(GF(2^31 - 1), ["x", "y"])
polys = [x^3 + y^2, x*y + x^2];

    and compute the Groebner basis

    using Groebner

basis = groebner(polys)
    3-element Vector{AbstractAlgebra.Generic.MPoly{AbstractAlgebra.GFElem{Int64}}}:
 y^3 + 2147483646*y^2
 x*y^2 + y^2
 x^2 + x*y

    We can also check that a set of polynomials forms a basis

    isgroebner(basis)
    true

    We check if x2y2+2x3y−xy2x^2y^2 + 2x^3y - xy^2x2y2+2x3y−xy2 is a member of ⟨x3+y2,xy+x2⟩\langle x^3 + y^2, xy + x^2 \rangle⟨x3+y2,xy+x2⟩, using the normalform function

    normalform(basis, x^2*y^2 + 2x^3*y - x*y^2)
    0

    with DynamicPolynomials

    We will compute the basis for the noon-2 system, arising from cellular network dynamics analysis [2]:

    using DynamicPolynomials

@polyvar x1 x2
system = [10*x1*x2^2 - 11*x1 + 10,
        10*x1^2*x2 - 11*x2 + 10]

groebner(system)
    3-element Vector{DynamicPolynomials.Polynomial{true, Int64}}:
 11x1² - 11x2² - 10x1 + 10x2
 110x2³ + 100x1x2 - 100x2² - 121x2 + 110
 10x1x2² - 11x1 + 10
